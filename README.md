# Aurora Book Store

## About the Project
Aurora Book Store is a literary e-commerce system designed to demonstrate proficiency in integrating Local Artificial Intelligence within corporate applications. Focus on advanced software engineering techniques, including Spring Boot, React, RAG, Tool Calling, and vector manipulation.

## Technologies Used
*   **Frontend**: React + UI Components (SPA for asynchronous consumption).
*   **Backend**: Java 21 + Spring Boot 3.x (Logic and API orchestrator).
*   **AI Framework**: Spring AI (Abstraction for calls, SSE, and Tools mapping).
*   **LLM Engine**: Ollama (Local execution environment).
*   **Models**: Llama 3.1 (8B) for reasoning and Nomic-embed-text for vectorization.
*   **Databases**: MySQL (Transactional) and ChromaDB (Vector/Semantic).
*   **Infrastructure**: Docker Compose (Container deployment).
